Proje Hakkında
Farklı gezegenlerde fizik deneylerini simüle eden konsol tabanlı C programı.
Geliştirici: SubayinH

Özellikler

9 farklı fizik deneyi
8 gezegen için simülasyon (Merkür - Neptün)
Pointer tabanlı dizi işlemleri
Ternary operator ile girdi kontrolü

Kurulum
Derleme:
bashgcc -o uzay_simulasyon main.c -lm
./uzay_simulasyon
Windows:
bashgcc -o uzay_simulasyon.exe main.c -lm
uzay_simulasyon.exe
Kullanım

Programı çalıştırın
Adınızı girin
Menüden deney seçin (1-9)
İstenen değerleri girin
Tüm gezegenler için sonuçları görün
Çıkmak için -1 girin
